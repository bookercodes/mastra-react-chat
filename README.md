# mastra-react-chat

Minimal example of a React frontend connecting to a Mastra agent with streaming.

- `client/` - React app using `@mastra/client-js`
- `server/` - Mastra agent with tools

## Setup

Create a `.env` file in `/server` with your OpenAI API key:

```bash
OPENAI_API_KEY=your-key-here
```

## Run

```bash
pnpm install
pnpm dev
```
